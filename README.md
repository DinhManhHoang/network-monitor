# C500 - DASHBOARD

Mã nguồn C500 - DASHBOARD, phần mềm của khóa luận 'Xây dựng hệ thống giám sát mạng dựa vào lời gọi hệ thống và dữ liệu luồng mạng' 

## Cài đặt

```
git clone https://github.com/DinhManhHoang/network-monitor.git
cd backend/
npm install
cd ..
cd frontend/
npm install
cd ..
npm run start
```

Truy cập vào địa chỉ https://localhost:8000/

## Lưu ý

Phần mềm cần có NodeJS và MongoDB để hoạt động:
* NodeJS: https://nodejs.org/en/
* MongoDB: https://www.mongodb.com/
